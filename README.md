# cho_minimini_project
# 📊 [Project] 유튜브 댓글 텍스트 분석을 통한 맛집 콘텐츠 마케팅 전략 도출

본 프로젝트는 맛집 추천 유튜브 채널의 댓글 데이터를 활용하여, 사용자 반응(Likes)과 콘텐츠 속성 간의 상관관계를 분석하고 **전환율이 높은 콘텐츠 기획을 위한 마케팅 인사이트**를 도출하는 것을 목표로 합니다.

---

## 🎯 1. 프로젝트 목적 (Business Objective)
* **사용자 니즈 파악**: 특정 카테고리(한식, 양식 등)에 따른 유저 반응도(좋아요 수) 분석
* **콘텐츠 최적화**: 댓글 길이와 반응도의 상관관계를 분석하여 효과적인 CTA(Call to Action) 및 소통 전략 수립
* **핵심 키워드 추출**: NLP(자연어 처리)를 통한 실제 고객의 관심 키워드 및 브랜드 이미지 정립

## 🛠 2. 사용 기술 (Tech Stack)
* **Language**: `Python`
* **Library**: `Pandas` (데이터 전처리), `Seaborn/Matplotlib` (시각화), `Konlpy` (형태소 분석)
* **Tool**: `Jupyter Notebook`

## 📈 3. 주요 분석 프로세스 및 결과

### ① 카테고리별 성과 분석 (Performance Marketing View)
* **분석 내용**: 한식, 중식, 일식, 양식 등 카테고리별 '좋아요' 합계 및 평균 집계
* **인사이트**: 어떤 음식 카테고리가 가장 높은 바이럴(Viral) 잠재력을 가졌는지 확인

### ② 댓글 길이와 반응도의 상관계수 분석 (User Engagement)
* **분석 방법**: `review_len` 컬럼 생성 후 `likes`와의 상관계수($r$) 산출
* **결과**: 산점도와 회귀선을 통해 '정성적인 긴 댓글'이 실제 커뮤니티 활성화에 기여하는지 검증

### ③ 텍스트 마이닝을 통한 관심 키워드 추출
* **분석 내용**: 명사 기반 워드클라우드 및 특정 키워드(예: '성수', '줄 서나요') 필터링
* **인사이트**: 유저들이 실제 방문을 고려할 때 가장 중요하게 생각하는 요소(위치, 대기 시간 등) 파악

## 💡 4. 마케팅 제언 (Strategic Insight)
* **데이터 기반 타겟팅**: 분석 결과 특정 카테고리의 반응이 월등할 경우, 해당 카테고리 중심의 'Short-form' 콘텐츠 확산 전략 제안
* **커뮤니티 가이드**: 댓글 피드백 분석을 통해 구독자가 질문하는 포인트(예: 주차, 가격)를 영상 내 자막으로 선제적 노출하여 정보 전달력 강화
